Russian Cyrillic Keyboard (Transliteration)

Type Romanized Russian (e.g., "privet" → "привет") for real-time Cyrillic output. Uses GOST standard. Supports digraphs like zh/sh/ch/shch. Copy the result.

GOST Romanization reference: a=а, e=е, i=и, o=о, u=у, y=ы, ya=я, ye=е, yo=ё, yu=ю; b=б, v=в, g=г, d=д, zh=ж, z=з, j=й, k=к, l=л, m=м, n=н, p=п, r=р, s=с, t=т, f=ф, kh=х, ts=ц, ch=ч, sh=ш, shch=щ, '=ь.
